🚀 Features
    • Add new todos
    • Edit and save toggle 
    • Delete todos
    • Mark todos as Active or Completed 
    • Filter by:
        ◦ Search text
        ◦ Hide completed
        ◦ Hide inactive
    • Persistent storage using localStorage
    • Dynamic DOM rendering after every state update

🧠 Technical Highlights
    • Event delegation for handling dynamic elements
    • Unique ID generation strategy to avoid duplication
    • Centralized render function to sync UI with state
    • JSON serialization/deserialization for persistent storage
    • Conditional rendering for edit mode

🛠 Tech Stack
    • HTML
    • CSS
    • Vanilla JavaScript (ES6+)
    • Browser localStorage API
